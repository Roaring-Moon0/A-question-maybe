# **App Name**: Heartfelt Unveiling

## Core Features:

- Interactive Introduction: A landing section with a soft animation and a call to action button to begin the heartfelt journey.
- Memory Input: An input field for users to enter a favorite memory, with a crayon-style underline animation when typing.
- Personality and Emotion Capture: Input fields to capture personality (single word) and emotion (emoji rating).
- Tone Selection: Users selects the preferred tone, using hand-drawn button to choose if message should be poetic, flirty or silly.
- Favorite Thing Input: Users types their favorite things. Includes soft animations and floating heart effects while the users types.
- AI Message Generation: AI tool using Gemini to generate a personalized romantic message based on the collected user input.
- Emotional Proposal Response: Emotional and playful 'Yes/No' response section with animated button interactions (dodging, confetti, heartfelt messages).
- Data persistence in Firestore: User inputs, the final message and responses (Yes/No) should all be stored and time-stamped.

## Style Guidelines:

- Primary color: Blush Pink (#FFB6C1) for a romantic and gentle feel.
- Background color: Desaturated rose (#FDF3F2) to set a soft, warm tone, reminiscent of a love letter.
- Accent color: Lavender Mist (#E8D6FF) for interactive elements, contrasting with the pink while maintaining a delicate feel.
- Headline font: 'Playfair', a serif font for headings, to provide a romantic and elegant feel.
- Body font: 'Poppins', a san-serif font, to offer clean and soft readability.
- Quote font: 'Caveat', to provide handwritten feel for quotes.
- Crayon-style icons for buttons and interactive elements, adding to the handcrafted aesthetic.
- Single-page design with smooth transitions between sections, creating a flowing, intimate experience.
- Subtle animations, such as parallax scrolling of floating hearts, hand-drawn hover effects, and a typewriter effect for the AI-generated quote.
